# GitHub Repository 생성 후 실행할 명령어들

# 1. GitHub Repository를 만든 후, 여기서 YOUR_USERNAME을 본인 계정으로 바꿔주세요
git remote add origin https://github.com/YOUR_USERNAME/aws-iot-electron-client.git

# 2. 코드 업로드
git push -u origin main

# 3. 빌드 파일들 확인
ls -la release/

# 4. GitHub에서 Release 생성 (웹에서 수동으로)
# - https://github.com/YOUR_USERNAME/aws-iot-electron-client/releases/new
# - Tag: v1.0.0
# - Title: AWS IoT Core 테스트 클라이언트 v1.0.0
# - Release 파일들 업로드:
#   - AWS IoT Core 테스트 클라이언트-1.0.0-arm64.dmg
#   - AWS IoT Core 테스트 클라이언트-1.0.0-x64.dmg  
#   - AWS IoT Core 테스트 클라이언트-1.0.0-arm64.zip
#   - AWS IoT Core 테스트 클라이언트-1.0.0-x64.zip

# 5. GitHub Pages 설정
# - Repository Settings → Pages
# - Source: Deploy from a branch
# - Branch: main / (root)
# - download-page.html을 index.html로 이름 변경

echo "GitHub Repository 주소를 알려주시면 바로 업로드하겠습니다!"
