📋 설치 및 실행 방법

  1. 다운로드: 위의 버튼에서 본인 Mac에 맞는 파일을 다운로드
  2. 설치: DMG 파일을 열고 Applications 폴더로 드래그하거나, ZIP을 압축 해제
  3. 실행: 앱을 더블클릭 (보안 경고 시 다시 한 번 클릭)
  4. 연결: AWS IoT 엔드포인트와 인증서 파일을 선택하여 연결
  5. 테스트: MQTT 메시지 발행/구독으로 연결 테스트 완료!
